# Data Professional Survey Dashboard

A comprehensive Power BI project that transforms raw survey responses from over 600 global data professionals into an interactive recruitment and salary insights dashboard. This project demonstrates end-to-end data processing—from messy Excel survey results to a polished, professional BI solution.

## 🚀 Project Overview
The goal of this project was to analyze the current landscape of the data industry, including salary trends, job satisfaction, and the difficulty of entry across different roles and regions.

### Key Features:
* **KPI Tracking:** Instant visibility into **Average Salary** and **Total Survey Participants** using Card visuals.
* **Difficulty Gauge:** A specialized Gauge Chart measuring the perceived **"Difficulty to Break Into Data."**
* **Geographic Analysis:** Dynamic filters to break down data by region (e.g., **United States**, **India**, etc.) to understand localized market trends.
* **Professional Styling:** Applied a custom theme and color scheme to ensure a unique and professional user experience.

---

## 🛠️ Technical Workflow

### 1. Data Transformation (Power Query)
The project utilizes a real-world survey dataset. To prepare it for visualization, I performed the following in **Power Query**:
* **Standardization:** Cleaned inconsistent naming conventions for job titles to ensure accurate grouping.
* **Data Cleaning:** Handled null values and removed duplicates from the raw survey responses.
* **Data Typing:** Ensured currency and numerical fields were properly formatted for KPIs and calculations.

### 2. Dashboard Development
* Built interactive visualizations to highlight key survey metrics.
* Implemented **Slicers** for regional filtering, allowing for deep-dive analysis into specific markets.
* Utilized **Card visuals** for high-level KPIs to provide an immediate snapshot of the data.

### 3. Finalization
* Designed the layout for maximum readability and visual impact.
* Applied a professional color palette to distinguish this report from standard templates.

## 📈 Insights Derived
* Identified the most common job titles and their corresponding salary brackets.
* Visualized the sentiment regarding the ease of entering the data field globally.
* Compared entry-level barriers and salary expectations across different global regions.
